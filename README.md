# GreenHouse - Houseplants Store

Assessment project implementing a small React + Redux store for houseplants.

How to run:

1. npm install
2. npm start

Notes:
- Add images to `public/assets` as described in the README there.
- This project uses React, React Router and Redux Toolkit.

GitHub repository: Please push this folder to your public GitHub repo to get the 2 points.
